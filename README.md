# python读取.json文件数据并进行计算
## 项目简介

实现了基于 Python 的高维向量坐标系计算工具，支持：

坐标系变换
向量投影
向量与坐标轴夹角
坐标系面积计算
支持任意维度

---

## 项目结构

```
linear_algebra_json/
│
├── data.json              # JSON数据
│
├── coordinate_system.py   # 坐标系类
├── task_processor.py      # 任务执行
├── main.py                # 主程序入口
│
├── README.md                  # 项目说明
└── requirements.txt           # 依赖           

```
