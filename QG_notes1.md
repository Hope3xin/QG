### **json库**

#### 1. JSON简介

##### 1.1 什么是 JSON

（JavaScript Object Notation）一种**轻量级数据交换格式（结构化字符串格式）**

特点：

基于文本（纯字符串）

跨语言（Python / Java / C++ / JS 等都支持）

##### 1.2 JSON的核心用途

1. 前后端数据传输（最常见）
2. 配置文件（如 `.json`）
3. 存储结构化数据
4. API 返回数据格式

#### 2. JSON语法规则

##### 2.1 基本规则

1. **键必须是字符串（必须双引号）**
2. 使用 `:` 表示键值对
3. 使用 `,` 分隔元素
4. 使用 `{}` 表示对象
5. 使用 `[]` 表示数组

#### 3.JSON中常用的方法

 json.dumps() —— Python 对象→ JSON字符串

常用参数：

| 参数         | 作用     |
| :----------- | -------- |
| indent       | 美化缩进 |
| ensure_ascii | 支持中文 |

json.loads() —— JSON字符串 → Python

 json.dump() —— 写入文件（Python → 文件）

json.load() —— 从文件读取（文件 → Python）

### **空间几何直观回顾**

#### 向量：

**是什么：**1.空间中的箭头（长度和方向）2.有序的数字列表3.结合以上两个观点

**向量加法**和**向量数乘**（线性代数围绕两种基本运算）：

|            | 物理观点                   | 列表观点       |
| ---------- | -------------------------- | -------------- |
| 向量的加法 | 运动                       | 对应项相加     |
| 向量的数乘 | 缩放（标量的作用就是缩放） | 分量与标量相乘 |

**线性组合**：

基向量： **i，j**被称作**基向量**，是**缩放的对象**（用数字描述向量时依赖于基向量）

线性组合：两个数乘向量的和

**张成的空间**：所有可以表示为给定向量**线性组合**的向量的**集合**

二维

| 给定的二维向量     | 张成的空间             |
| ------------------ | ---------------------- |
| 一般的两个二维向量 | 所有二维向量的集合     |
| 两个共线的向量     | 一条直线上的向量的集合 |
| 两个零向量         | 一个点                 |

三维

| 给定的向量                               | 张成的空间                         |
| ---------------------------------------- | ---------------------------------- |
| 一般的两个向量                           | 过原点的平面                       |
| 三个向量，第三个落在前两个所张成的平面上 | 平面                               |
| 一般的三个向量                           | 空间中所有的三维向量（平面的扫动） |

**线性相关**：一个向量可以表示为其他向量的线性组合。（有多个向量时，可以移除之一而不减小张成的空间）反之，所有向量都给张成的空间增加新的维度，称它们是**线性无关**的。

**向量空间的一组基**（严格定义）：张成该空间的一个**线性无关**的向量集



#### 矩阵与线性变换

**变换(运动观点)**：输入向量移动到输出向量的位置

**线性变换****（对空间的挤压伸展）**：①**任意**直线→直线②原点不变（保持网格线平行且等距分布）

用数值描述线性变换，考虑基向量**i，j**的变换，可以确定任意向量的变换；

**二维线性变换**：仅由四个数字完全确定，即基向量的变换。几何直观上在变换后的基向量基础上进行运动。



#### 矩阵乘法与线性变换复合

**两个矩阵相乘几何意义：**两个线性变换相继作用（对**i，j**变换两次）

矩阵乘法交换律不成立：变换先后顺序不同，最终位置不同

矩阵乘法结合律成立：变换顺序不变

**三维**：3*3矩阵描述**i，j，k**的线性变换

**点乘**：一向量投影*一向量长度（与顺序无关）

**叉乘**：

首先，定义一个三维空间到数轴的线性变换

然后，考虑这个变换的对偶向量

①应用这个变换和与对偶向量点乘等价

②从几何角度思考，这个对偶向量必然与两个给定向量垂直，长度等于两个向量张成的平行四边形面积

**行列式：**

二维：对平面的缩放比例（大小、方向、维度的变化）

三维：对体积的缩放比例
