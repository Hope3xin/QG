# 模型评估

## 实验结果

线性回归

- 均方误差 MSE：0.3952
- 决定系数 R²：0.4279

（估值）

逻辑回归

- 分类准确率：86.25%

## 分析

1. 线性回归能一定程度预测红酒质量，但 R² 不高，说明数据本身非线性较强。
2. 逻辑回归分类效果较好，准确率超过 86%，说明好坏酒的特征差异明显。



## 相同点

1. 都是线性模型
2. 都使用**梯度下降**优化参数
3. 梯度计算公式形式一致

## 不同点

| 项目     | 线性回归           | 逻辑回归               |
| -------- | ------------------ | ---------------------- |
| 任务类型 | 回归（预测连续值） | 二分类（预测类别概率） |
| 输出范围 | (−∞,+∞)            | （0,1） 概率           |
| 激活函数 | 无                 | Sigmoid                |
| 损失函数 | 均方误差 MSE       | 交叉熵损失             |
| 标签类型 | 连续数值           | 离散 0/1               |
| 评估指标 | MSE、R²            | 准确率                 |
